Endpoints

Search
  1. Pass in a single-word search term after the /search/{searchTerm}
  2. Receive a JSON response of search results with this type:
    {
    title: "Sample Title",
    imageUrl: "url",
    pageUrl: "url",
    topic: "Sample Topic",
    views: "# Views",
    posted: "Date Posted"
    }
  3. Optional parameters are "max", "offset", and "format." "Format" defaults to JSON. User format returns an HTML page.
    Sample URL: /search/poodle?max=100&offset=50&format=user
Recent
  1. Hit the recent searches endpoint /recent
  2. Receive a JSON response of recent searches
    {
    searchTerm: "aSearchTerm"
    date: unix
    }
